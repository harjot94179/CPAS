Covid Prediction and Assistance System is designed to help you through these tough times.This web application focuses on creating a Covid prediction System which also guides the user in case he is found to be infected by the Coronavirus. In such case, this system provides user with a written guide which has necessary steps to be followed if user is found Covid positive, enlisting the steps required to help himself be protected in an environment where if he is found negative on a manual or physical diagnosis can stay protected from direct or indirect contact with any infected person.

Our Prespective

No ‘one can answer those questions but we can be take care of ourselves so in addition to the preventions and precautions one take we would like to add something to that in terms of a prediction and assistance system which will guide the prospective user by running an algorithm and comparing his body temperature and other symptoms with that of someone infected with Covid and if the results of the diagnosis came out to be positive then guide the user to what precautions he should take and alert the nearby hospitals and medical workers about a potent patient and route him to nearest pharmaceutical dispensaries.
